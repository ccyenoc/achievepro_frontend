# 🚀 AchievePro Frontend

A modern KPI tracking and management system frontend built with React.  
This application enables managers and staff to monitor performance, assign KPIs, and visualize progress through interactive dashboards.

---

## 📌 Features

- 📊 KPI Dashboard with real-time progress visualization  
- 👥 Role-based interface (Manager & Staff)  
- 📝 KPI assignment and tracking  
- ⚠️ Status indicators (At Risk, Underperform)  
- 🔍 Dynamic filtering and search  
- 📈 Interactive charts (Recharts)  
- 🎨 Responsive UI using Bootstrap  

---

## 🧱 Tech Stack

- **React** (Frontend framework)  
- **Bootstrap** (Responsive layout)  
- **Recharts** (Data visualization)  
- **React Router** (Navigation)
